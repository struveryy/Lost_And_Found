# 🔍 FindIt — Smart College Lost & Found Platform

A modern and intelligent Lost & Found Web Application built using HTML, CSS, and JavaScript to help students quickly report, discover, and recover lost belongings across campus through smart matching and community-driven features.

✨ Features
✅ Report Lost Items
✅ Post Found Items
✅ Smart Match Detection 🔗
✅ Search & Filter System 🔍
✅ Interactive Item Cards 📦
✅ Leaderboard & Hero Points 🏆
✅ Achievement & Badge System 🎖️
✅ User Profile Dashboard 👤
✅ Toast Notifications 🔔
✅ Responsive Modern UI 📱💻
✅ Animated Modals & Smooth UX ✨
✅ Campus Community Engagement 🤝

🛠️ Tech Stack
**HTML5 • CSS3 • JavaScript**

🚀 Purpose
Designed to create a smarter and safer campus environment by helping students reconnect with lost belongings quickly through a modern digital platform.
